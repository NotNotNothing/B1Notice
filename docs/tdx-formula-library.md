# 通达信公式库功能说明

## 功能概述

通达信公式库功能允许用户保存、管理和快速调用多个选股公式，方便在不同市场环境下切换使用不同的选股策略。

## 主要功能

### 1. 公式管理

- **创建公式**：点击"新建公式"按钮，填写公式名称、公式内容和说明
- **编辑公式**：点击公式卡片上的编辑按钮修改已有公式
- **删除公式**：点击删除按钮移除不再使用的公式
- **设为默认**：将常用公式设置为默认，方便快速选择

### 2. 公式使用

- **快速应用**：在收盘选股界面点击"公式库"按钮，选择要使用的公式
- **实时预览**：公式库显示当前正在使用的公式，避免混淆
- **语法验证**：保存时自动验证公式语法，确保公式可用

### 3. 公式测试

- **自选股测试**：点击"用自选股测试"按钮，在自选股中验证公式效果
- **结果查看**：查看测试结果，包括命中股票数量、详细信息等
- **错误提示**：显示公式执行过程中的错误信息

## 使用步骤

### 首次使用

1. 进入"收盘选股"页面
2. 选择"通达信公式"模式
3. 点击"公式库"按钮打开公式库面板
4. 点击"新建公式"创建第一个公式

### 创建公式

1. 填写公式名称（必填）：例如"BBI金叉选股"
2. 填写公式内容（必填）：例如 `XG: CROSS(C, BBI) AND J < 20;`
3. 填写公式说明（可选）：例如"筛选收盘价金叉BBI且J值小于20的股票"
4. 选择是否设为默认公式
5. 点击"保存"

### 使用公式

1. 在公式库中找到要使用的公式
2. 点击"使用此公式"按钮
3. 公式会自动填充到选股配置中
4. 点击"保存规则"保存配置
5. 点击"手动执行"或等待自动执行

### 测试公式

1. 在选股配置界面输入或选择公式
2. 点击"用自选股测试"按钮
3. 查看测试结果：
   - 命中数量：显示有多少只自选股符合条件
   - 命中列表：显示具体的股票信息和指标
   - 错误信息：如果有错误，会显示详细错误信息

## 支持的变量和函数

### 变量

- **价格相关**：C/CLOSE/P（收盘价）、O/OPEN（开盘价）、H/HIGH（最高价）、L/LOW（最低价）
- **成交量**：V/VOL/VOLUME（成交量）
- **技术指标**：K/D/J（日线KDJ）、BBI（多空指标）、WJ/WEEKLYJ（周线J值）
- **自定义指标**：VOLRATIO（量比）、ABOVEBBIDAYS（连续站上BBI天数）、BELOWBBIDAYS（连续跌破BBI天数）、CHANGEPCT（涨跌幅）

### 函数

- **引用函数**：REF（引用历史数据）
- **移动平均**：MA（简单移动平均）、EMA（指数移动平均）、SMA（加权移动平均）
- **极值函数**：HHV（最高值）、LLV（最低值）
- **数学函数**：ABS（绝对值）、MAX（最大值）、MIN（最小值）
- **逻辑函数**：IF（条件判断）
- **统计函数**：COUNT（计数）、EVERY（全部满足）、EXIST（存在满足）
- **交叉函数**：CROSS（金叉）

### 示例公式

#### 1. BBI金叉选股
```
XG: CROSS(C, BBI) AND J < 20 AND VOLRATIO > 1.2;
```
说明：收盘价金叉BBI，J值小于20，量比大于1.2

#### 2. KDJ底部反转
```
XG: CROSS(K, D) AND K < 20 AND J < 10;
```
说明：K线金叉D线，且都在超卖区域

#### 3. 多头排列选股
```
MA5 := MA(C, 5);
MA10 := MA(C, 10);
MA20 := MA(C, 20);
XG: MA5 > MA10 AND MA10 > MA20 AND C > MA5;
```
说明：5日线>10日线>20日线，且收盘价站上5日线

## 数据库表结构

### TdxFormula 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 公式唯一标识 |
| userId | String | 所属用户ID |
| name | String | 公式名称 |
| formula | String | 公式内容 |
| description | String | 公式说明 |
| isDefault | Boolean | 是否为默认公式 |
| createdAt | DateTime | 创建时间 |
| updatedAt | DateTime | 更新时间 |

## API 接口

### 获取公式列表
```
GET /api/tdx-formulas
```

### 创建公式
```
POST /api/tdx-formulas
Body: { name, formula, description?, isDefault? }
```

### 获取单个公式
```
GET /api/tdx-formulas/:id
```

### 更新公式
```
PUT /api/tdx-formulas/:id
Body: { name, formula, description?, isDefault? }
```

### 删除公式
```
DELETE /api/tdx-formulas/:id
```

## 注意事项

1. **公式语法**：必须符合通达信公式语法规范
2. **公式验证**：保存时会自动验证语法，错误公式无法保存
3. **默认公式**：每个用户只能有一个默认公式，设置新默认会自动取消旧默认
4. **公式命名**：同一用户的公式名称不能重复
5. **数据准备**：执行选股前确保已有当日市场数据快照
